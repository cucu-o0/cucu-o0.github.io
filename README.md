# Vero's Portfolio

🌐 Live at [cucu-o0.github.io](https://cucu-o0.github.io)

## Run locally

Make sure you're using Homebrew Ruby, then serve with Jekyll:

```bash
export PATH="/opt/homebrew/opt/ruby/bin:$PATH"
which ruby  # should now show /opt/homebrew/opt/ruby/bin/ruby
bundle exec jekyll serve --livereload
```

## Introduction

Portfolio based on a Minimal Jekyll theme for GitHub Pages.
